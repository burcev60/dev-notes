# 🐍 Python

Раздел посвящён языку Python: стандартным библиотекам, встроенным функциям, типизации, ООП паттернам и асинхронности.

## 📂 Подразделы

| Раздел | Описание |
|--------|----------|
| [`std-libraries/`](std-libraries/README.md) | Стандартные библиотеки: os, sys, collections, itertools, datetime, json, functools, typing, logging, re, contextlib |
| [`built-ins/`](built-ins/README.md) | Встроенные функции: map, filter, zip, enumerate, sorted и др. |
| [`type-hints/`](type-hints/README.md) | Аннотации типов: основы, дженерики, протоколы |
| [`oop-patterns/`](oop-patterns/README.md) | ООП: классы, наследование, декораторы, итераторы, dataclasses |
| [`async/`](async/README.md) | Асинхронность: asyncio, async/await, конкурентное выполнение |
